## 1. エグゼクティブサマリー（3行）

1. 競合を「人間↔AI」×「単発鑑定↔日常習慣」でマッピングすると、**「AI × 日常習慣」の象限に確立プレイヤーが不在**であり、これが [01_Market_Research.md](./01_Market_Research.md) §9 で示した空白地帯とそのまま重なる。
2. 直接競合（占い各社）より脅威なのは間接競合——**ChatGPTへの人生相談・16Personalities的な診断文化・メンタルケアアプリ**——であり、ユーザーの「悩みを整理したい時間」を奪い合う相手として設計に織り込む。
3. 差別化は機能ではなく構造で行う: 毎日習慣化・恐怖訴求しない倫理ブランド・シェアされる出力・関係性占いの構造的バイラル・AI検索先行・LINE起点CRM・価格透明性の**7つの勝ち筋**を本ドキュメントで確定する。

## 2. 競合マップ（2軸ポジショニング）

**軸の定義（判定文）**
- 横軸「人間 ↔ AI」: 鑑定の主要供給者。人間鑑定師が売上の過半を担うなら左、アルゴリズム/AIなら右
- 縦軸「単発鑑定 ↔ 日常習慣」: ユーザーの中央値的な利用頻度。月1回以下なら下、週4回以上の接触設計なら上

```
                          日常習慣（毎日〜週数回）
                                   │
        メディア型占い              │        Co-Star / The Pattern
      （しいたけ占い・             │       （海外・日本語未対応が実質）
        星ひとみ的メディア）        │
                                   │        16Personalities等の診断
        LINE占いの無料コンテンツ    │       （習慣というより文化・語彙）
                                   │
                                   │      ★ 本サービス
                                   │      【AI × 日常習慣 × 対話】
                                   │       ← この象限に確立プレイヤー不在
  人間 ──────────────────────────┼──────────────────────────── AI
                                   │
        電話占い                    │        各社AI占いチャット
      （ヴェルニ・ピュアリ等）      │       （単発の鑑定ツール型が主流）
                                   │
        チャット占い                │        汎用LLM占い
      （LINE占い・ウラナ・          │       （ChatGPT/DeepSeek算命:
        ココナラ占い）              │        習慣導線なし・都度プロンプト）
                                   │
        対面鑑定・占いの館          │
                                   │
                          単発鑑定（悩んだ時だけ）
```

**マップの読み方（結論）**
- 左下（人間×単発）は最も混雑し、最もARPPUが高く、最も批判リスクが高い。**本サービスはここで戦わない**
- 左上（人間×習慣）はメディア型占いが強いが、一方通行で対話・パーソナライズがない
- 右下（AI×単発）は乱立しているが、単発ツールは再訪理由がなくLTVが構造的に低い
- **右上（AI×習慣）が本サービスの陣地**。ここを最速で要塞化する（勝ち筋1: 毎日習慣化）

## 3. 競合9カテゴリ分析

各カテゴリを共通フォーマット（代表サービス/強み/弱み/価格帯/UX/ブランド/口コミ状況/継続構造/課金導線/SNS戦略/本サービスへの示唆）で分析する。数値・内情はすべて推定・公開情報ベース。

### 3.1 カテゴリ(1): AI占い（専業AI占いチャット・汎用LLM利用）

| 観点 | 内容 |
|---|---|
| 代表 | 各社のAI占いチャット（大手占いメディアのAI鑑定機能、スタートアップのAIタロット等）、汎用LLMの占い利用（ChatGPTへの相談、DeepSeek算命的なプロンプト共有文化） |
| 強み | 即時・24時間・低価格または無料。心理障壁が低い（人間に話す恥ずかしさがない）。汎用LLMは会話品質が高く無料 |
| 弱み | 専業AI占いの多くは「1問1答の自販機」型で対話の深さ・記憶・習慣導線がない。汎用LLMは命式計算を誤る・出力が不安定・占い専用の世界観/儀式性がない・履歴が占い文脈で蓄積されない |
| 価格帯 | 無料〜1回数百円、月額数百〜千円台が中心（推定） |
| UX | チャットUIだが「引いたカードの演出」「結果の保存・見返し」が弱いものが多い |
| ブランド | 総じて弱い。「AIだから安い・お試し」の位置づけで、指名されるブランドが未確立 |
| 口コミ状況 | 「意外と当たる」「無料で遊べる」のライトな言及が中心。熱量の高いファンダムは未形成（推定） |
| 継続構造 | ほぼなし。プッシュ・デイリーコンテンツ・ストリーク等の習慣装置が未実装のものが大半 |
| 課金導線 | 回数制限→解放型のフリーミアムが主流。転換率は低いと推定 |
| SNS戦略 | ほぼ皆無か、運用が形骸化 |
| **本サービスへの示唆** | 最大の学び: **「AI占い」という機能だけでは差別化にならない**（すでにコモディティ）。勝負は (a) 計算エンジンの正確性（計算はコード、解釈はAI）、(b) ユーザーの文脈記憶、(c) 習慣導線、(d) 出力の美しさ、(e) ブランド。特に汎用LLMとの差別化は「毎日届く・覚えている・占いの儀式性がある・シェアできる」の4点で判定する |

### 3.2 カテゴリ(2): 電話占い（ヴェルニ、ピュアリ 等）

| 観点 | 内容 |
|---|---|
| 代表 | 電話占いヴェルニ、ピュアリ、ウィル、カリス等の大手プラットフォーム |
| 強み | 圧倒的に高いARPPU（1分200〜400円超×平均20〜30分と推定）。深い悩みに人間が応える満足度。人気鑑定師の指名リピート構造。SEO（「復縁 占い」等の悩みキーワード）に強い |
| 弱み | 高価格による利用障壁と後悔リスク。待ち時間（人気鑑定師は予約困難）。品質分散。依存・高額請求への社会的批判が慢性化。若年層には電話という手段自体が重い |
| 価格帯 | 1分200〜500円程度（推定）。月数万円以上使うヘビーユーザーが収益の柱と言われる |
| UX | 電話 or 予約制。心理的・時間的な摩擦が大きい。初回無料分（10分等）で導入 |
| ブランド | 「よく当たる鑑定師在籍」型の的中訴求。広告色が強く、若年層のブランド好感は低いと推定 |
| 口コミ状況 | 「当たった」称賛と「使いすぎた・依存した」後悔が二極化。アフィリエイトレビューが検索結果を占拠 |
| 継続構造 | 鑑定師への個人的依存（担当制）。恋愛の悩みが解決しない限り続く、という**悩みの未解決性に依存した継続** |
| 課金導線 | 分課金＋先払いポイント。初回無料→不安が最も高まった瞬間に課金が発生する設計 |
| SNS戦略 | 弱い。獲得はSEO・アフィリエイト・リスティング中心 |
| **本サービスへの示唆** | (1) 深い悩み相談の需要と支払意思の大きさを実証している——本サービスの従量課金（深掘り鑑定チケット）はこの需要を1/10以下の価格で受ける。(2) 「悩みの未解決性に依存した継続」は本サービスの対極。本サービスは「解決に向かうから続けたい」継続を作る（判定: NPS 30→50、Spine §5）。(3) 彼らのSEO牙城（悩み系KW）はドメージ済み市場だが、AI検索（AEO）は未防衛——先行機会（勝ち筋5） |

### 3.3 カテゴリ(3): チャット占い（LINE占い、ウラナ 等）

| 観点 | 内容 |
|---|---|
| 代表 | LINE占い（チャット/電話鑑定＋占いコンテンツの複合）、ウラナ、ココナラ占い（スキルマーケット型） |
| 強み | LINE占いは**LINEという日常導線上にある**ことが決定的優位。テキストなので電話より心理障壁が低い。ココナラは価格の幅広さ（数百円〜）とレビュー透明性 |
| 弱み | チャット鑑定は応答待ち時間が発生（人間供給）。1文字/1分単位の課金が「引き延ばし」への不信を生みやすい。コンテンツ型占いは買い切りで継続構造が弱い |
| 価格帯 | チャット1分100〜200円程度、コンテンツ1件数百〜2,000円程度（推定） |
| UX | LINE上で完結する手軽さは高評価。ただし鑑定師選びの認知負荷が高い（数千人から選ぶ） |
| ブランド | LINE占いはプラットフォームブランドで安心感があるが、「LINE=占い」の想起は弱い |
| 口コミ状況 | 手軽さの好評と、料金の不透明感（思ったより高かった）への不満が混在（推定） |
| 継続構造 | 鑑定師リピート＋日次の無料コンテンツ（おみくじ的機能）で接触は作っているが、対話の習慣化には未到達 |
| 課金導線 | コイン/ポイント先払い制。無料コンテンツ→有料鑑定への送客 |
| SNS戦略 | 公式コンテンツ配信中心。UGC設計は弱い |
| **本サービスへの示唆** | (1) LINEを日常導線として使う戦術は最重要の学び——本サービスはLINE公式アカウント/ミニアプリをCRMの背骨にする（勝ち筋6、→ [11_CRM.md](./11_CRM.md)）。(2) 「数千人から選ぶ」認知負荷は、AI単一人格（選ばなくていい）の逆張り機会。(3) コイン制の価格不透明感への不満は、価格透明性（勝ち筋7）の根拠 |

### 3.4 カテゴリ(4): 診断アプリ・診断コンテンツ（16Personalities、診断メーカー 等）

| 観点 | 内容 |
|---|---|
| 代表 | 16Personalities（MBTI系）、診断メーカー、各種メディアの心理テスト、ストレングス系無料診断 |
| 強み | **バイラル構造が全カテゴリ中最強**。結果=アイデンティティのラベルで、投稿・プロフィール記載・会話ネタとして自走拡散。登録不要・無料・3分の摩擦ゼロ導線 |
| 弱み | 収益化が弱い（広告・一部有料レポート）。1回きりで継続構造が皆無。結果が静的でパーソナライズの深さがない |
| 価格帯 | 無料中心。有料レポートが数千円（推定） |
| UX | 質問→結果→シェアの一直線。結果ページのデザイン（タイプカラー・キャラ化）が投稿映えする |
| ブランド | 16PersonalitiesはZ世代の共通語彙になるレベルの文化的浸透。ただし「サービス」ではなく「テスト」として認知 |
| 口コミ状況 | プロフィール記載・「私INFPだから」的な日常会話への浸透。UGCが無限に再生産 |
| 継続構造 | なし（再診断は年1回あるかないか） |
| 課金導線 | 結果の一部ロック→詳細レポート販売 |
| SNS戦略 | 戦略というよりユーザーが勝手に広める構造。結果画像のシェア最適化のみ |
| **本サービスへの示唆** | (1) **獲得エンジンとして診断構造を丸ごと取り込む**: 無料・登録不要・3分・タイプ化された結果・投稿映え、を新規獲得の入口に置く（→ [08_Viral_Strategy.md](./08_Viral_Strategy.md)）。(2) 彼らの弱点（1回きり・収益化難）を、診断→毎日の運勢→対話→サブスクの導線で埋める。(3) 判定: 診断入口経由の新規のD7継続率を全体D7目標（25%、Spine §5）と別建てで計測し、20%以上を合格線とする |

### 3.5 カテゴリ(5): 恋愛アプリ（Pairs 等マッチングアプリ）

| 観点 | 内容 |
|---|---|
| 代表 | Pairs、with、タップル等。特にwithは心理テスト・性格診断を恋愛マッチングに組み込んだ先行例 |
| 強み | 「恋愛の意思決定」という占い最大ジョブの隣接領域で、巨大な課金市場（国内マッチングアプリ市場は推定数百億〜1,000億円規模と言われる）を確立。月額サブスクの定着 |
| 弱み | 出会いの後（関係の進展・迷い・不安）はサポート外。ここが占いのジョブ領域 |
| 価格帯 | 月額3,000〜4,000円程度（男性課金中心、推定） |
| UX | スワイプ・マッチの完成されたループ |
| ブランド | 「真剣な出会い」路線でマス化に成功 |
| 口コミ状況 | 利用の公言が一般化（かつてのスティグマは解消傾向） |
| 継続構造 | 目的達成（交際）で卒業するチャーン構造が宿命 |
| 課金導線 | フリーミアム→月額。女性無料モデル等のセグメント別価格 |
| SNS戦略 | TVCM・縦型動画広告の大量投下型 |
| **本サービスへの示唆** | (1) 競合というより**ジョブの上流/下流の隣接プレイヤー**。マッチング後の「相手の気持ちが分からない」「進展の迷い」は本サービスの相性占い・恋愛相談が受けるジョブ。(2) withの成功は「診断×恋愛」の相性の良さを実証——相性占いの需要の裏付け。(3) 月額3,000円台の恋愛関連支出が成立している事実は、Standard ¥2,980（Spine §4）の価格妥当性の傍証。(4) 将来の提携候補（相性コンテンツ提供）としても検討 |

### 3.6 カテゴリ(6): 自己分析アプリ・ツール（ミイダス、ストレングスファインダー 等）

| 観点 | 内容 |
|---|---|
| 代表 | ミイダス（可能性診断）、ストレングスファインダー（CliftonStrengths）、キャリア系適性診断、グッドポイント診断等 |
| 強み | 「自分の強みを知る」ジョブをビジネス文脈の信頼性（アセスメント・統計）で提供。企業・採用文脈の収益源が太い。書籍連動（ストレングスファインダー）の権威性 |

| 価格帯 | 無料（リード獲得型）〜数千円（書籍・レポート） |
| UX | 質問数が多く途中離脱が多いと推定。結果は詳細だが活用導線が弱い |
| ブランド | 「科学的・ビジネス的」。占いとの差別化を明確に打ち出す側 |
| 口コミ状況 | キャリア文脈・就活文脈での言及。エンタメ的拡散は少ない |
| 継続構造 | ほぼなし |
| 課金導線 | B2B/採用送客が本丸。C向け課金は補助的 |
| **SNS戦略** | 弱い（B2B寄りのため） |
| **本サービスへの示唆** | (1) 「自己理解」ジョブには**科学の顔**と**物語の顔**の2つの入口があり、本サービスは物語側（四柱推命・占星術による自己記述）を取る。20〜30代男性の拡張ターゲット（Spine §8）には「自己分析」の語彙で届けるのが有効。(2) 彼らの弱点「結果が一度きり」に対し、本サービスは「日々の出来事×あなたの性質」の継続解説で差別化。(3) キャリア相談テーマは男性ユーザー獲得の入口として専用導線を作る |

### 3.7 カテゴリ(7): メンタルケアアプリ（Awarefy、muute 等）

| 観点 | 内容 |
|---|---|
| 代表 | Awarefy（AIメンタルパートナー）、muute（AIジャーナリング）、emol等。海外ではCalm、Headspace、Woebot系 |
| 強み | 「感情の言語化・整理」という本サービスと**同一ジョブ**を、CBT（認知行動療法）等の科学的裏付けで提供。企業提携（福利厚生）ルートを持つ。継続利用（毎日の記録）を設計済み |
| 弱み | 「メンタルケアアプリを使う」こと自体の心理障壁（自分は病んでいる、という自己認知が必要）。エンタメ性・偶然性がなく飽きやすい。無料層の広がりが占いより狭い |
| 価格帯 | 月額1,000〜2,000円前後のサブスクが中心(推定） |
| UX | ジャーナリング・気分記録・AIコメント。落ち着いたトーン |
| ブランド | 誠実・科学的。マス向けの華やかさはない |
| 口コミ状況 | 良質だが静か。バイラル性は低い |
| 継続構造 | 日次記録・通知・振り返り機能で本格的に設計されている（本カテゴリの見習うべき点） |
| 課金導線 | フリーミアム→サブスク。機能制限型 |
| SNS戦略 | コンテンツマーケ・専門家監修記事中心 |
| **本サービスへの示唆** | (1) **最重要の間接競合**。本サービスの実体（感情整理の習慣）が最も近い。ただし彼らは「ケアが必要な人」の入口しか持たず、本サービスは「占い」というエンタメの入口で**同じジョブを心理障壁ゼロで届けられる**——これが構造優位。(2) 夜のふりかえり・気分記録・ストリーク等のリテンション装置は彼らの設計から学ぶ（→ [07_UX_Improvement.md](./07_UX_Improvement.md)）。(3) 判定: 「占いを入口に来て、実質ジャーナリング的に使っているユーザー」比率を計測し、リテンションコホートとして追う |

### 3.8 カテゴリ(8): SNS占いコミュニティ・占い師個人アカウント（しいたけ占い、星ひとみ的メディア型 等）

| 観点 | 内容 |

## 4. 「本当の競合は占いではない」分析

ユーザーのジョブ（不安の整理・意思決定の後押し・聞いてもらう）を起点にすると、時間と悩みを奪い合う相手は占いカテゴリの外にいる。

### 4.1 ジョブ別の真の競合

| ユーザーの瞬間（ジョブ） | 現在の解決手段（真の競合） | その手段の弱点 | 本サービスの勝ち方（判定文） |
|---|---|---|---|
| 深夜、不安で眠れない | X/TikTokの無限スクロール、ChatGPTに相談 | スクロールは不安を増幅する。ChatGPTは有能だが「儀式性・意味づけ・毎日の続き」がない | 「夜のふりかえり鑑定」を22〜26時のプッシュで定着させる。判定: 深夜帯（22-26時）セッション比率とその翌朝再訪率 |
| 彼の気持ちが分からない | 友人へのLINE相談、恋愛系まとめ・知恵袋検索 | 友人には回数の限界と気まずさがある。検索結果は一般論 | 相手の情報を入れた相性鑑定＋「今日の相手との接し方」。判定: 相性鑑定の週次リピート率 |
| 仕事を辞めるか迷う | 転職サイトの診断、同僚への相談、ChatGPT | 転職サイトはポジショントーク。ChatGPTは論理整理はできるが「背中を押す物語」がない | 「あなたの時期」の物語化＋選択肢の構造整理。判定: キャリアテーマ鑑定の完了率と保存率 |
| 自分が何者か知りたい | 16Personalities、自己分析本 | 静的で一度きり。日々の出来事と接続しない | 命式ベースの自己分析＋日々の出来事への当てはめ解説。判定: プロフィール保存率・自己分析系鑑定のシェア率 |
| ただ聞いてほしい | 友人、家族、（一部）カウンセリング、ChatGPT | 友人・家族は利害と評価が絡む。カウンセリングは高額で敷居が高い | 非審判の傾聴＋占いの語彙による意味づけ。判定: 傾聴系セッションの平均ターン数とWRS化率 |
| 暇つぶし・遊び | 診断メーカー、ゲーム、ショート動画 | 消費して終わり。自分の物語に残らない | 遊べる診断入口→保存される「自分のページ」への接続。判定: 診断入口経由のD7継続率20%以上 |

### 4.2 最大の競合 = ChatGPT（汎用LLM）への態度決定

- **事実認識**: 「人生相談ならChatGPTで無料でできる」は本サービスへの最頻の反論であり、最大の代替圧力である
- **戦略決定**: 汎用LLMと「相談の賢さ」で正面競争しない。差別化は次の5点に固定する
  1. **儀式性と世界観**: カードを引く・星を読むという「意味づけの演出」（ChatGPTは道具であり、儀式ではない）
  2. **正確な占術エンジン**: 命式・ホロスコープ計算をコードで担保（汎用LLMは計算を誤る、→ [01_Market_Research.md](./01_Market_Research.md) §6.5）
  3. **記憶と物語の継続**: 過去の相談・運勢の推移を「あなたの物語」として蓄積・参照
  4. **向こうから届く**: 毎朝の運勢プッシュ。ChatGPTは自分から開かない限り無
  5. **シェアできる出力**: 美しい結果カード。ChatGPTの回答はシェアする形をしていない
- **判定**: ユーザーアンケートで「ChatGPTではなく本サービスを使う理由」の上位3つが上記5点のいずれかに該当する状態を維持する（四半期調査）

**クロスSWOTの結論**
- **S×O（勝負手）**: S1+S3×O1 → 「毎日寄り添うAI」の要塞化を最優先。S2×O3 → 倫理を明文化してブランドの中核に（守りではなく攻め）
- **W×O（補強）**: W1×O2 → 認知はAI検索とUGCで作る（広告で買わない）。W5×O4 → 「AI占いは安っぽい」を出力の美しさと正確な占術エンジンで反証
- **S×T（防衛）**: S3+S4×T1 → 汎用LLM差別化5点（§4.2）の徹底。W4×T2 → 大手が動く前の12ヶ月で習慣・データ・引用シェアの先行者資産を積む
- **W×T（回避）**: W3×T4 → 出力監査体制（[01](./01_Market_Research.md) §4.2）でカテゴリ炎上の巻き添えを回避し、むしろ「安全な方のAI占い」として逆張り訴求
