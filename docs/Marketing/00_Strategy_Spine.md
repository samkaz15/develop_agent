## 1. 事業定義

- **プロダクト**: AI占いサービス（MVP稼働済み）。AIとの対話型鑑定（タロット・西洋占星術・四柱推命等）を、Web / LINE / アプリで提供する
- **提供価値の本質**: 「当たるかどうか」ではなく **「不安が言語化され、心が軽くなり、明日の一歩が決まる体験」** を売る。カテゴリは占いだが、ジョブは「意思決定の後押し」と「感情の整理」
- **ブランド名**: 未定（`09_Brand_Strategy.md` で命名方針を定義。それまで全文書で「本サービス」と表記）

## 2. 最終目標（North Star Goal）

- 月商 **50億円**（年商600億円）にスケール可能な事業構造
- 日本一の「継続率・口コミ発生率・LTV・広告効率（CAC）」
- **広告費に依存しないグロースモデル**（有料広告経由の新規獲得は全体の20%以下を維持）

## 3. North Star Metric（NSM）

> **WRS: Weekly Resonant Sessions（週次共鳴セッション数）**
> = 1週間に「鑑定完了 かつ（保存 / シェア / メモ追記 / 再訪問のいずれか）」まで到達したセッション数

- 「開いた」ではなく「心が動いた」を計測する
- WRSは継続率・口コミ率・課金率すべての先行指標として機能する
- 補助NSM: **D30継続率** と **紹介経由新規比率**

## 4. 収益モデル（50億円/月の分解）

| レバー | 内容 | 目標値 |
|---|---|---|
| サブスク | 月額プラン（Light ¥980 / Standard ¥2,980 / Premium ¥9,800） | 収益の60% |
| 従量課金 | チケット制の深掘り鑑定・特別鑑定 | 収益の25% |
| 物販・その他 | 鑑定書PDF・グッズ・提携・API | 収益の15% |

**月商50億円の成立条件（Standard中心のブレンドARPPU ¥3,000と仮定）**

```
月商50億円 = 課金者 約167万人 × ARPPU ¥3,000
           = MAU 約1,700万人 × 課金率10% × ARPPU ¥3,000
```

- 日本の占い利用経験者は成人の約5〜7割と推定され、MAU 1,700万人は「日本の占い関心層の約1/3」に相当する。国内のみでは上限に近いため、**36ヶ月目以降に多言語展開（繁体字圏・東南アジア・英語圏）を前提**とする

## 6. グロースモデル（4つのループ）

広告ではなく、以下の4つの自己強化ループで成長する。

1. **コンテンツループ（SEO/AEO/LLMO）**: 鑑定テーマ×悩みの構造化コンテンツ → 検索・AI検索から流入 → 鑑定体験 → 検索需要データがコンテンツに還元
2. **UGCループ（シェア設計）**: 鑑定結果の「シェアしたくなる出力」→ SNS投稿 → 閲覧者が自分も試す → 新規UGC
3. **リファラルループ（関係性占い）**: 相性占い・友達占いが構造的に他者を巻き込む → 招待 → 招待された側も相性を占う
4. **リテンションループ（習慣化）**: 毎朝の運勢 → 夜のふりかえり → ストリーク → 翌朝再訪 → LTV増 → プロダクト再投資

## 7. 基本思想（優先順位の原則）

1. 広告で解決する前に、プロダクト改善で解決する
2. CAC投下より継続率改善（D30 +1%は広告費数千万円に相当する）
3. 「シェアされる体験」は機能ではなく出力の美しさと言葉から生まれる
4. LTVを伸ばさない施策は実行しない
5. すべての施策はICE/RICEで優先度を判定し、実験として設計・計測する（→ `15_Experiment_Backlog.md`）

## 8. ターゲット（要約）

- **コア**: 20〜34歳女性。恋愛・仕事・自己理解の意思決定に迷いを抱え、SNS（Instagram/TikTok/X）で占い・診断コンテンツに日常的に接触している層
- **拡張**: 35〜49歳女性（結婚・キャリア・家族）、20〜30代男性（キャリア・自己分析）、Z世代（診断遊び・推し活文脈）
- 詳細は `03_Persona.md`

## 9. ブランドの核（要約）

- **世界観**: 「夜空の下で、いちばん話を聞いてくれる存在」— 断定で脅さない、寄り添って背中を押す
- **トーン**: あたたかい・具体的・脅さない・押しつけない。「不安を煽って課金させる」旧来型占いの逆を行く
- **倫理**: 恐怖訴求・依存誘発・断定的な不幸予言は行わない（これ自体が最大の差別化でありブランド資産）
- 詳細は `09_Brand_Strategy.md`
